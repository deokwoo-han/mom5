# 🕯️ AI 솔빙 스트레스: LAMP 마스터 (Worry Eraser)

**"걱정이 많은 사람을 위한 심리학 수업"**의 LAMP 이론을 기반으로 설계된 AI 심리 케어 애플리케이션입니다.
Google Gemini(Generative AI)를 활용하여 사용자의 불안을 분석하고, 객관화하며, 마음챙김(Mindfulness)을 돕는 도구입니다.

## 📌 주요 기능

이 앱은 LAMP 커리큘럼에 따라 4단계의 심리 치유 과정을 제공합니다.

### 1️⃣ 1단계: 걱정 이름표 붙이기 (Labeling)

* **상황과 생각 포착:** 걱정되는 상황을 기록하고, 그것이 '실제적인 걱정'인지 '가상의 걱정'인지 구분합니다.
* **감정 및 신체 반응 기록:** 현재 느끼는 감정(불안, 우울, 분노 등)과 신체 감각을 구체화합니다.
* **AI 객관적 시선(거리두기):** Gemini AI가 제3자의 관찰자 시점이 되어, 사용자의 상황을 건조하고 객관적인 사실로 분리해줍니다.

### 2️⃣ 2단계: AI 심리 분석 (Analysis)

* **불안 패턴 모니터링:** 기록된 불안 농도를 시각화(Line Chart)하여 감정의 변화를 추적합니다.
* **종합 심리 리포트:** 축적된 기록을 바탕으로 AI가 메타 걱정(걱정에 대한 걱정), 통제 욕구, 인지적 오류 등을 분석하고 맞춤형 행동 처방을 제공합니다.

### 3️⃣ 3단계: 관계 테라피 (Communication)

* **나 전달법(I-Message) 연습:** 갈등 상황에서 사실(Fact), 감정(Emotion), 요청(Request)을 분리하여 입력합니다.
* **AI 대화 코칭:** 입력된 내용을 바탕으로 상대방을 비난하지 않으면서도 단호하게 의사를 전달하는 세련된 대화법으로 변환해줍니다.

### 4️⃣ 4단계: 이완과 멈춤 (SOS)

* **4-7-8 호흡법 가이드:** 불안한 뇌를 진정시키는 호흡 훈련을 시각적 가이드와 함께 제공합니다.
* **점진적 근육 이완법:** 신체의 긴장을 줬다 푸는 과정을 통해 마음의 긴장을 완화합니다.
* **긴급 도움:** 자살 예방 및 위기 상담 전화번호 정보를 제공합니다.

## 🛠️ 기술 스택 (Tech Stack)

* **Language:** Python 3.8+
* **Framework:** [Streamlit](https://streamlit.io/) (Web UI)
* **AI Model:** [Google Generative AI (Gemini)](https://ai.google.dev/)
* **Data Manipulation:** Pandas

## 🚀 설치 및 실행 방법 (Getting Started)

### 1. 사전 준비

Google Gemini API 키가 필요합니다. [Google AI Studio](https://aistudio.google.com/app/apikey)에서 발급받으세요.

### 2. 라이브러리 설치

터미널에서 아래 명령어를 입력하여 필요한 패키지를 설치합니다.

```bash
pip install streamlit pandas google-generativeai

```

### 3. 애플리케이션 실행

프로젝트 폴더에서 아래 명령어를 실행합니다.

```bash
streamlit run mom5.py

```

### 4. 사용 방법

1. 웹 브라우저가 열리면 좌측 사이드바에 **Google Gemini API Key**를 입력합니다.
2. 사용할 AI 모델(Flash 또는 Pro)을 선택합니다.
3. 사이드바 메뉴에서 원하는 LAMP 단계를 선택하여 마음 챙김을 시작합니다.
